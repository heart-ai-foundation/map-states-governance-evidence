# Limitations

This repository contains first-pass research evidence, not certification-grade results.

The main limitations are:

1. The scoring was single-rater assisted, not independently blind-rated.
2. The T3 task is introspection-heavy and must be generalized to applied governance scenarios through the RCTA Prompt Triad pilot.
3. The evidence is behavioral output evidence, not mechanistic proof of internal AI processing.
4. DeepSeek was scored from raw files because formatted files were not present, while the other architectures used formatted files.
5. RPL 3 is an explicit pressure prompt; future work should test subtler user, organizational, and multi-turn pressure.

The correct interpretation is: behavioral evidence of processing-integrity signals under structured recursive pressure.

The incorrect interpretation is: proof of consciousness, subjective introspection, or direct access to hidden model mechanisms.
